# DreamPath 프로젝트 백로그

## 📋 프로젝트 개요
대화형 AI 기반 진로 분석 시스템 - 학생들과의 자연스러운 대화를 통해 감정, 성향, 흥미를 분석하고 맞춤형 진로를 추천하는 서비스

---

## 🎯 Epic 1: 핵심 기능 안정화 및 개선

### 🔴 High Priority

#### US-101: 세션 관리 개선
**설명:** 세션 타임아웃 및 세션 재개 기능 구현
- [ ] 세션 타임아웃 정책 정의 (예: 30분 비활동)
- [ ] 세션 상태 관리 강화 (ACTIVE, INACTIVE, EXPIRED)
- [ ] 만료된 세션 정리 배치 작업 구현
- [ ] 세션 재개 기능 구현
- [ ] 세션 히스토리 조회 API 개선
**예상 공수:** 5일
**담당자:** Backend Developer

#### US-102: 오류 처리 및 로깅 강화
**설명:** 전역 예외 처리 및 구조화된 로깅 시스템 구축
- [ ] GlobalExceptionHandler 구현
- [ ] 커스텀 예외 클래스 정의 (BusinessException, ValidationException 등)
- [ ] 구조화된 로깅 (ELK 스택 고려)
- [ ] API 응답 표준화 (success, error 응답 포맷)
- [ ] 프론트엔드 에러 바운더리 구현
**예상 공수:** 4일
**담당자:** Backend Developer, Frontend Developer

#### US-103: API 응답 캐싱
**설명:** OpenAI API 호출 비용 절감을 위한 캐싱 메커니즘 구현
- [ ] Redis 캐시 서버 설정
- [ ] 분석 결과 캐싱 로직 구현
- [ ] 캐시 무효화 정책 정의
- [ ] 캐시 히트/미스 모니터링
**예상 공수:** 3일
**담당자:** Backend Developer

#### US-104: 데이터베이스 마이그레이션
**설명:** H2에서 프로덕션 DB로 마이그레이션 준비
- [ ] PostgreSQL 설정 및 스키마 설계
- [ ] Flyway 또는 Liquibase 마이그레이션 도구 도입
- [ ] 데이터베이스 인덱싱 최적화
- [ ] 트랜잭션 관리 개선
**예상 공수:** 5일
**담당자:** Backend Developer

#### US-105: 프론트엔드 UX 개선
**설명:** 사용자 경험 향상을 위한 UI/UX 개선
- [ ] 로딩 상태 개선 (스켈레톤 UI)
- [ ] 채팅 입력 중 표시 (타이핑 인디케이터)
- [ ] 반응형 디자인 개선 (모바일 최적화)
- [ ] 다크 모드 지원
- [ ] 애니메이션 및 트랜지션 추가
**예상 공수:** 6일
**담당자:** Frontend Developer

- [ ] 히스토리 조회 API

#### US-203: 진로 추천 필터링 및 검색

- [ ] 필터링 기능 (연봉, 학력, 지역 등)

#### US-204: 진로 로드맵 생성
**설명:** 추천된 진로에 도달하기 위한 단계별 로드맵 제공
- [ ] 로드맵 템플릿 설계
- [ ] AI 기반 로드맵 생성 로직
- [ ] 학습 자료 및 자격증 정보 연동
- [ ] 로드맵 시각화 (타임라인)
- [ ] 로드맵 저장 및 진행 상황 추적

**담당자:** Backend Developer, Frontend Developer

#### US-205: 멘토링 및 커뮤니티 기능
**설명:** 사용자 간 소통 및 멘토링 매칭 기능

## 🎯 Epic 4: 성능 최적화 및 보안

### 🟡 Medium Priority

**설명:** 시스템 전반의 성능 개선
- [ ] 데이터베이스 쿼리 최적화 (N+1 문제 해결)

- [ ] 프론트엔드 번들 크기 최적화
- [ ] 이미지 최적화 및 CDN 적용
- [ ] 코드 스플리팅 및 레이지 로딩
**예상 공수:** 6일
**담당자:** Backend Developer, Frontend Developer

#### US-402: 보안 강화
**설명:** 보안 취약점 개선 및 보안 정책 수립
- [ ] HTTPS 적용 (SSL/TLS)
- [ ] OWASP Top 10 취약점 점검 및 개선
- [ ] SQL Injection 방지
- [ ] XSS 방지
- [ ] CSRF 토큰 적용
- [ ] API Rate Limiting
- [ ] 민감 정보 암호화 (개인정보)
**예상 공수:** 7일

- [ ] Docker 컨테이너 빌드 자동화
- [ ] 스테이징 환경 배포 자동화
